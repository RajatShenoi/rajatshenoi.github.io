Steps to install the game on windows

Step 1: Download Zip

To download the zip file from the HOME PAGE for the appropriate game.

Step 2: Extract the Zip file

To Extract the file all you need to do is open the zip and copy and paste the folder inside of it somewhere outside of the zip file.
NOTE: You need to copy paste the whole folder and not the files inside of it.

Step3: Run the app

Now open the folder and double click on the gamename.exe file.
You may be given a warning. In that case, you need to click more info, or the appropriate alternative and click on run anyway. Note, that you are getting this message as you are not installing from Windows Store / Microsoft Store, be assured that the file is safe to install.

Installation is Complete!
